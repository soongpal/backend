# 숭팔이

## 1. 프로젝트 개요

숭팔이는 **숭실대학교 학생들을 위한 공동구매 및 중고거래 플랫폼**입니다.  
<br>
사용자는 공동구매 참여자를 모집하거나 중고 물품을 거래할 수 있으며,  
채팅 기능을 통해 거래 및 공동구매 참여자 간 실시간 소통이 가능합니다.

<br>

- **공동구매** : 참여자 간 다중 채팅 지원
- **중고거래** : 판매자와 구매자 간 1:1 채팅 지원 <br><br>
- 개발기간 : 2025.04 ~ 2025.11  
- 팀구성 : 백엔드 2명, 프론트엔드 1명
<br>

## 2. 기술 스택

### Backend
- Spring Boot 3
- Java 21
- Spring Data JPA
- MySQL 8
- WebSocket (채팅 기능 구현)

### Infrastructure
- AWS EC2
- AWS RDS

### Documentation
- Swagger (API 문서화)


## 3. 주요 기능

### 공동구매
- 공동구매 게시글 생성
- 참여자 간 **다중 채팅 기능**

### 중고거래
- 중고 거래 게시글 생성
- 판매자와 구매자 간 **1:1 채팅 기능**

### 사용자 기능
- 회원가입 및 로그인
- 마이페이지
- 게시글 좋아요 기능
